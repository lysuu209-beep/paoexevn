⚠️ TẠI SAO BẢN CŨ KHÔNG GỬI ĐƯỢC EMAIL THẬT?
Browser chặn kết nối SMTP trực tiếp vì lý do bảo mật — bản cũ chỉ giả lập (simulate).

✅ V3 dùng EmailJS — gửi email thật qua browser, miễn phí 200 email/tháng

3 bước setup:
1. Đăng ký tại emailjs.com (miễn phí)
2. Email Services → Add New Service → Gmail → lấy Service ID
3. Email Templates → Create Template → thêm {{to_email}} {{subject}} {{message}} → lấy Template ID
4. Account → General → lấy Public Key
